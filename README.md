# SocialBook 📱

A modern, responsive, and pixel-perfect Facebook clone built using pure **HTML5** and **CSS3**.

Created by **Kishore**!

---

## ✨ Features

- **Responsive Design:** Fully responsive layout optimized for Desktop, Tablet, and Mobile views.
  - *Desktop:* Full 3-column layout (Left Sidebar, Main Post Feed, Right Sidebar).
  - *Tablet (max-width: 900px):* Automatically collapses sidebars to focus purely on the main feed.
  - *Mobile (max-width: 600px):* Shrinks search input and expands navigation to fit smaller screens.
- **Modern Sticky Navbar:** Features a sticky navigation bar with search bar, navigation shortcuts, and online status user profile icon.
- **Left Sidebar:** Includes navigation links to important pages (Latest News, Friends, Groups, Marketplace, Watch) and developer shortcuts.
- **Interactive Story Section:** Features a stylized horizontal story gallery with upload status and member stories.
- **Create Post Card:** A clean interface to write new posts, complete with options for Live Video, Photo/Video, and Feelings/Activity.
- **Rich Post Feed:** Multiple post containers showcasing rich text, custom tags, and dynamic images alongside interactive activity metrics (Like, Comment, Share count).
- **Right Sidebar:** Displays upcoming events, real-time advertisements, and active online connections.
- **Polished Typography:** Uses Times New Roman and elegant hover effects for link interaction.

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser (like Google Chrome, Mozilla Firefox, Microsoft Edge, or Safari) to view this project locally.

### How to Run Locally

1.  Download or clone the files to your local machine.
2.  Navigate to the project directory.
3.  Double-click `index.html` or open it with your favorite browser.

## 📤 How to Push to Git

To push this project to your GitHub repository, follow these steps in your terminal (Git Bash, Command Prompt, or PowerShell):

### 1. Initialize Git Repository
In your project folder `SocailBook`, run:
```bash
git init
```

### 2. Stage All Project Files
Add all the files in the directory to git staging:
```bash
git add .
```

### 3. Commit the Changes
Create your first commit:
```bash
git commit -m "Initial commit: SocialBook Facebook clone using HTML and CSS"
```

### 4. Link and Push to your GitHub Repository
Go to [GitHub](https://github.com) and create a new repository named `SocialBook`. Do not check the boxes to add a README, `.gitignore`, or License.

Once created, run the following commands in your terminal (make sure to replace the placeholder URL with your actual repository URL):
```bash
# Rename the default branch to main
git branch -M main

# Add your GitHub repository as remote origin
git remote add origin https://github.com/<your-username>/SocialBook.git

# Push your code to GitHub
git push -u origin main
```
