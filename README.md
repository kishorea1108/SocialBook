# SocialBook 📱

A modern, responsive, and pixel-perfect Facebook clone built using pure **HTML5** and **CSS3**.

Created by **Kishore**!

---

## ✨ Features

- **Responsive Design:** Fully responsive layout optimized for Desktop, Tablet, and Mobile views.
  - _Desktop:_ Full 3-column layout (Left Sidebar, Main Post Feed, Right Sidebar).
  - _Tablet (max-width: 900px):_ Automatically collapses sidebars to focus purely on the main feed.
  - _Mobile (max-width: 600px):_ Shrinks search input and expands navigation to fit smaller screens.
- **Modern Sticky Navbar:** Features a sticky navigation bar with search bar, navigation shortcuts, and online status user profile icon.
- **Left Sidebar:** Includes navigation links to important pages (Latest News, Friends, Groups, Marketplace, Watch) and developer shortcuts.
- **Interactive Story Section:** Features a stylized horizontal story gallery with upload status and member stories.
- **Create Post Card:** A clean interface to write new posts, complete with options for Live Video, Photo/Video, and Feelings/Activity.
- **Rich Post Feed:** Multiple post containers showcasing rich text, custom tags, and dynamic images alongside interactive activity metrics (Like, Comment, Share count).
- **Right Sidebar:** Displays upcoming events, real-time advertisements, and active online connections.
- **Polished Typography:** Uses Times New Roman and elegant hover effects for link interaction.

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser (like Google Chrome, Mozilla Firefox, Microsoft Edge, or Safari) to view this project locally.

### How to Run Locally

1.  Download or clone the files to your local machine.
2.  Navigate to the project directory.
3.  Double-click `index.html` or open it with your favorite browser.
